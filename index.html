New Relic Safari Extension

Download this project as a .zip file Download this project as a tar.gz file

Setup Instructions

  1. Log in to your New Relic dashboard and click on the application you want to setup.
  2. Extract the application and account ID's from the URL: https://rpm.newrelic.com/accounts//applications/
  3. API key can be found at: Account Settings > Api + Web Integration > API Access
  4. Enter your credentials and hit Save.
